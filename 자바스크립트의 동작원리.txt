# Javascript

자바스크립트의 동작원리가 뭐임? V

자바스크립트 코드가 어떻게 실행되고, 메모리에 어떻게 저장되며, 어떤 순서로 처리되는지에 대한 것

메모리 힙(Memory Heap) : 메모리 할당이 일어나는 곳

콜 스택(Call Stack) : 호출 스택이 쌓이는 곳

처음에 동작원리에 대한 내 생각

1. 동작원리? 그딴거 왜 알아야함? 그런거 몰라도 부트캠프 수업과정 Todolist,계산기 다 구현했는데?

그럼 동작원리를 알아야 하는가? V

자바스크립트의 동작원리를 알아야 하는 이유 

1. 성능상의 문제해결이나 예외처리를 하는 방법을 모를 수 있음.
2. 간단한 기능구현은 괜찮겠지만 대규모 프로젝트에서는 내부 원리를모르면 효율적으로 만들거나 유지보수하기가 어려움 
3. 협업할 때 다른 개발자들이 이해하기 어려운 코드가 나올 가능성이 큼.

내 공부방식과 비교 V 

실습위주 
ㄴ 문제점 : 너무 실습만하면 내부 동작원리를 배우기 부족함, -> 메모리 영역을 파악하기 어렵기 때문

그럼 어떻게 고쳐볼 수 있는가? V 

실습이 나쁘다는건 아님. 실습을 해보며 메모리 구조를 조금이라도 시각화 하여 파악해보는게 좋을것 같음
